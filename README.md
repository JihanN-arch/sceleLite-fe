# SceleLite - FE 💻

Repositori ini adalah antarmuka pengguna (UI) dari SceleLite.

## 🎯 Fitur Utama (Frontend)
- **Dynamic Dashboard:** Tampilan berbeda untuk Dosen (Manage Course) dan Mahasiswa (Enrollment).
- **Smart Axios Interceptor:** Menangani penyisipan token otomatis dan *Silent Refresh Token* jika sesi habis tanpa logout paksa.
- **Real-time Validation:** Validasi kapasitas kursus dan bentrok SKS sebelum dikirim ke server.
- **Responsive UI:** Didesain menggunakan Tailwind CSS agar nyaman dibuka di berbagai ukuran layar.

## 🛠️ Tech Stack
- React.js (Vite)
- Tailwind CSS
- Axios (API Client)
- Lucide React (Icons)
- React Hot Toast (Notifications)
